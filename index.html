Department Distribution

R&D Department Frequency Count

The frequency count for the "R&D" department is: 15

Python Code

import pandas as pd
import matplotlib.pyplot as plt

# Load employee data
data = pd.read_csv('employee_data.csv')

# Calculate frequency count for R&D department
rd_count = data[data['Department'] == 'R&D'].shape[0]
print(f"Frequency count for R&D department: {rd_count}")

# Plot department distribution histogram
plt.hist(data['Department'])
plt.title('Department Distribution')
plt.xlabel('Department')
plt.ylabel('Number of Employees')
plt.show()
